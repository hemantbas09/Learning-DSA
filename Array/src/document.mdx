6. Types of Arrays:

# single dimensional arrays: linear list of elements. eg int []number={1,2,3}
# multi dimensional Arrays: arrays inside the array like matrices. eg int numbers={{1,2},{3,4},{5,6}}
# Jagged Arrays: array which have different length row:eg int numbers={{1, 2}, {3, 4, 5}, {6}}


# 2-D Array:

Initialization:
int[][] matrix = {{1, 2, 3}, {4, 5, 6}};

using new:
int[][] matrix = new int[2][3]; // 2 rows, 3 columns, all 0s
matrix[0][1] = 10;             // Sets row 0, column 1 to 10

Accessing:

System.out.println(matrix[1][2]); // Prints 6 from {{1, 2, 3}, {4, 5, 6}}

# Length Operation in Arrays:
